Today, NetHydra proudly announces the release of NetHydra 2026.2 — one of the biggest updates ever introduced to the NetHydra ecosystem.

This release delivers major improvements across the core system, performance enhancements, updated tooling, expanded compatibility, and a variety of internal changes designed to improve both flexibility and usability for security professionals.

Before jumping into [What's new](#whats-new), let us introduce [SONAR](/doc/Introduction/wiki-sonar).

NetHydra [SONAR](/doc/Introduction/wiki-sonar) is a dedicated enterprise-focused edition of NetHydra, built for organizations that require a more flexible security stack compared to the standard edition, NetHydra Express.

[SONAR](/doc/Introduction/wiki-sonar) includes additional tools and components focused on several areas of cybersecurity operations, including:

- Web Application Security
- Threat Intelligence
- Defensive Security Operations
- Security Assessment & Analysis

The goal of [SONAR](/doc/Introduction/wiki-sonar) is to provide a more adaptable platform for enterprise environments, allowing teams to customize and scale their workflows based on operational requirements.

In addition to the new [SONAR](/doc/Introduction/wiki-sonar) variant, NetHydra 2026.2 also includes various improvements to tool integration, and overall platform experience.

This release represents another major step forward for the NetHydra project as we continue building a modern, flexible, and security-focused platform for researchers, defenders, and enterprise organization.

_let's jump to [What's new?](#whats-new)_

## What's new —

- [Firefox Theme](#firefox-theme) - Revert compat Firefox theme
- [Core system improvements](#system-changes) - Added zram as default configuration
- [New tools](#new-tools) - 4 tools added
- [Page update](#pages-update) - NetHydra page update

## Firefox Theme

In this update, we reverted the Firefox theme to a more compatible and classic design inspired by the 2020 Firefox appearance.

Special thanks to [@BGMP](https://github.com/BGMP/firefox-revert-proton) for making this possible.

![NetHydra classic firefox theme style](/assets/graphics/post_graphics/nethydra-2026.02/compat-classic-firefox.png)

## System Changes

In this update, we decided to enable `zram` as the default configuration on NetHydra to provide a better overall system experience and improved memory management.

## New Tools

As with every NetHydra release, this update also introduces several new tools and package additions — including tools specifically added for the [SONAR](/doc/Introduction/wiki-sonar) edition.

Here is a quick rundown of some of the new tools included in NetHydra 2026.2:

- [MISP](/doc/tools/sonar-tools-misp) - Open source threat intelligence and information sharing platform.
- [Burp Suite](/doc/tools/sonar-tools-burpsuite) - Comprehensive penetration testing platform designed for web application security testing.
- [Nuclei](/doc/tools/sonar-tools-nuclei) - Fast and customizable vulnerability scanner based on templates.
- [CyberChef](/doc/tools/sonar-tools-cyberchef) - The Cyber Swiss Army Knife web app for encryption, encoding, compression and data analysis

Several of these tools are included as part of the expanded [SONAR](/doc/Introduction/wiki-sonar) toolkit to support enterprise-focused security workflows, threat analysis, and defensive operations.

## Get NetHydra 2026.2

**Fresh images?**: what do you waiting for? Grab now! [Get NetHydra](/get/)

too lazy to get Fresh Images? Roll it now!

```
──(nethydra@NetHydra-Express)─[~]
└─── $ sudo apt update && sudo apt full-upgrade -y
[...]

──(nethydra@NetHydra-Express)─[~]
└─── $ [ -f /var/run/reboot-required ] && sudo reboot -f
```

You should now running NetHydra 2026.2:

```
──(nethydra@NetHydra-Express)─[~]
└─── $ grep VERSION /etc/os-release
VERSION="2026.2"
VERSION_ID="2026.2"
VERSION_CODENAME="sana"

──(nethydra@NetHydra-Express)─[~]
└─── $ uname -r
6.12.86+nethydra-rt-amd64
```

Or if you're using [SONAR](/doc/sonar) edition:

```
──(nethydra@NetHydra-Express)─[~]

VERSION="2026.2"

VERSION_CODENAME="sana"

──(nethydra@NetHydra-Express)─[~]
```
